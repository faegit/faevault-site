# FAEVault 3.6.4

发布日期：2026-07-24

## Windows

- WebDAV 关联改为输入远端文件夹地址，自动补全保险库文件名，与 Android 端操作一致。
- 自动同步支持云端硬盘 + WebDAV 双目标独立配置与独立调度。
- 已有远端文件时新增"下载覆盖本地"选项，从远端全量替换本地保险库。
- 云端同步对话框改用按标签页实际内容自动调整高度。
- 云端硬盘与 WebDAV 标签页内边距收紧、间距统一，防止控件向下拉伸。
- 认证成功后自动隐藏 WebDAV 认证表单区域。
- 修复云端硬盘 / WebDAV 关联后界面未及时刷新按钮状态的问题。
- 修复 WebDAV 下载覆盖本地后提示"关联异常"的问题。
- 远端文件存在时可选同步合并、上传覆盖或下载覆盖。

## Android

- WebDAV 创建目录开关默认关闭。
- 自动同步支持云端硬盘 + WebDAV 双目标独立配置、独立调度与独立失败计数。
- 云端同步页面上传覆盖 / 下载覆盖 / 重新关联 / 取消关联按钮收至"更多操作"可折叠区域。
- WebDAV 上传改用临时文件 + 原子移动策略，冲突时回退直写并清理临时文件。
- WebDAV 冲突检测增加字节级 SHA-256 兜底。
- 修复自动同步失败三次后正确暂停对应目标而非整体禁用。
- 修复 3.6.3 → 3.6.4 升级时自动同步 per-target 配置的向后兼容。
- 修复 If-Match 头对非 HTTP 标准 ETag 的支持。
