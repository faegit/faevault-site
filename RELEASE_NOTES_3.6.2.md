# FAEVault 3.6.2

发布日期：2026-07-22

## Windows

- “关于”页面新增在线检查更新，可展示最新正式版本说明并下载对应 Windows 安装包。
- 新增自定义标题栏、原生窗口圆角与更完整的窗口交互。
- 条目详情页可查看和下载附件，覆盖安全笔记与自定义模块。
- 浏览器私有 IP 自动填充新增即时授权入口。
- 安全扫描流程与 Android 统一，修复进度突进和重复完成提示。
- 修复云端逻辑内容一致时仍误报远端变更、类目排序崩溃及清空回收站后列表不刷新的问题。
- 安装器改为简体中文，默认安装到 `C:\Program Files\vault`，并可识别和卸载旧版以避免文件冲突。

## Android

- “关于”页面新增在线检查更新，自动选择适合设备架构的 APK。
- 条目详情页新增附件信息、完整性状态与安全导出操作。
- 修复动态码卡片在列表重建或内存回收后只显示标题的问题。
- 完善 Passkey 冲突隔离、一次性请求令牌和异常详情警告。
- 统一中英文通知、设置、附件和紧急恢复密钥文案。

## 安装

完整步骤见 [安装与升级指南](./INSTALLATION.md)，其中包含 Windows、Android、Chrome、Edge 和 Firefox 浏览器扩展教程。
